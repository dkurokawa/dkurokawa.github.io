Portman v1.4.0

ポート調停ツール。vibe codingでAIが使ってくれると楽できます。Claude code検証済み。

インストール

# Go install
go install github.com/daikurokawa/portman/cmd/portman@latest

# または、リポジトリをクローンしてビルド
git clone https://github.com/daikurokawa/portman.git
cd portman/portman-go
go build -o portman ./cmd/portman

使い方

# ポートを取得/割り当て
portman get "project-name"

# ポート一覧
portman list

# ダッシュボード
portman dashboard

詳細は GitHub を参照してください。

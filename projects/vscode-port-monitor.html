VSCode Port Monitor v0.5.1

VS Codeでシステム開発時に関連するポートを登録し利用状況を表示

インストール

VS Code Marketplaceで「Port Monitor」を検索してインストール

機能

🔍 マルチポート監視

複数のホストとポートを同時に監視し、リアルタイムで状態を更新

🏷️ インテリジェント設定

ポート範囲の自動展開、Well-knownポート名のサポート

🛑 プロセス管理

ポートを使用中のプロセスを特定し、安全に終了可能

使い方

{
  "portMonitor.hosts": {
    "localhost": [3000, 3001, 3002]
  }
}

詳細は GitHub を参照してください。
